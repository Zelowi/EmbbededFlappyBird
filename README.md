# EmbbededFlappyBird


Bare-metal Flappy Bird game made on STM32F407 Discovery board using a 0.96" SSD1306 OLED display over I2C.

Features
Custom SSD1306 OLED driver
I2C communication written from scratch
Frame buffer graphics rendering
Moving pipes with random gaps
Bird movement and gravity
Collision detection
Button input handling
Game loop implementation
Bare-metal STM32 register-level programming
Hardware
STM32F407 Discovery board
SSD1306 128x64 OLED display (I2C)
Push button
Breadboard and jumper wires
Technologies
C
STM32
Embedded Systems
I2C
GPIO
OLED graphics
Bare-metal programming

What I Learned

During this project I learned:

How I2C communication works internally
OLED display memory organization
Frame buffer rendering
Embedded debugging using STM32CubeIDE debugger
Working with timers and delays
Structuring embedded projects
Collision detection logic
Hardware/software debugging
